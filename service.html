Описание

Версия для слабовидящих на сайтах государственных учреждений, муниципальных медицинских и образовательных организаций необходима в обязательном порядке с 1 января 2016 года! (читать статью). Это требование регламентировано Федеральным законом N419-ФЗ «О внесении изменений в отдельные законодательные акты Российской Федерации по вопросам социальной защиты инвалидов в связи с ратификацией конвенции о правах инвалидов».

Использование модуля установки на сайт версии для слабовидящих категории АА за 9900 рублей. Категория АА - это полная доступность на сайте информации для слабовидящих, что подходит для большинства сайтов и полностью соответствует Федеральному закону РФ N419-ФЗ.
Использование модуля установки на сайт версии для слабовидящих категории ААА за 15900 рублей. Категория ААА рекомендуется для специализированных информационных ресурсов для людей с ограничениями по зрению.
Использование отраслевых готовых решений, уже включающих в себя версию сайта для слабовидящих категории АА и полностью соответствующих законодательству РФ и ГОСТ Р 52872-2012 «Интернет-ресурсы».
